######CONTROLS######

Up - Accelerate
Down - Brake
Left - Turn left
Right - Turn right
Space - Jump (Watch for the bounce!)

How to run:
just build in eclipse and run it.
If the games is too hard, here is a gameplay video:
https://vimeo.com/79712999

Algorithms used:
We use a bezier patch to draw and calculate collisions on the tunnels.
Other collision detection is mostly box collision detection.
The gravity drags the ship down until it hits a surface using a function called ontop.

Models used:
F14d - Right Hemisphere, LTD

Code used:
Skybox class found here:
http://code.google.com/p/tgra-project3/source/browse/trunk/src/is/ru/tgra/Skybox.java
